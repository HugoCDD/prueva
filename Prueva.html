En esta practica veremos los que son los strings lo cual son: cadenas de textos.
Para abrir una cadena de textos usamos el comando new string("x") pero tambien se puede solo usar dos comillas o comillas simples,
con estas tres formas podemos abrir una cadena de texto, agregando length podemos saber con cuantos
catacteres cuenta nuestra cadena de texto.

>

Métodos

  • ncadena hace referencia a cadena de texto.
  • ncadena.toUpperCase() nos sirve para poner una cadena de texto en mayusculas.
  • ncadena.tolowercase() nos sirve para poner una cadena de texto en minusculas
  • El comando: ncadena.includes("palabra") sirve para buscar si se encuentra una palabra en la cadena de texto
  • ncadena.trim() sirve para quitar los espacios en blanco de una cadena de texto
  • ncadena.split(" espacio ") este metodo dirve para agregar algo en los espacios en blanco de una cadena

Template string


En este caso veremos lo que es template strings:

los template strings nos sirven para poder unir cadenas de textos con variables por ejemplo: let saludo= `Hola mi nombre es ${nombre} ${apellido}.`; de esta manera estariamos mezclando la cadena de caracteres con variables que son nombre y apellido
